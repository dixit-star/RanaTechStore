##  Features

-  **Dynamic Product Listing**
  - Products are displayed using JavaScript
  - Each product includes image, title, and price

-  **Add to Cart Functionality**
  - Users can add products to the cart
  - Cart updates instantly without page reload

-  **Dynamic Cart Counter**
  - Cart icon shows real-time item count
  - Implemented using JavaScript DOM manipulation

-  **Automatic Price Calculation**
  - Total cart price is calculated dynamically
  - Updates automatically when items are added or removed

-  **Remove Items from Cart**
  - Users can remove individual products from the cart
  - UI updates instantly

-  **Responsive Design**
  - Fully responsive for mobile, tablet, and desktop
  - Built using CSS media queries

-  **Clean & User-Friendly UI**
  - Simple navigation
  - Beginner-friendly layout

-  **Fast Performance**
  - Built with pure HTML, CSS, and JavaScript
  - No frameworks or external libraries used
